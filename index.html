About This Project

This expanded project demonstrates my growing interest in environmental engineering at a national scale. By integrating real-time APIs from across America, I've created a comprehensive tool that visualizes environmental data patterns from coast to coast. This USA-wide dashboard showcases advanced coding skills, API integration, and data visualization while providing practical environmental monitoring for any major US city. It represents the evolution from local environmental awareness to understanding national environmental challenges and solutions.

About EcoMap USA

A comprehensive national environmental monitoring dashboard covering major cities across the United States. Features real-time API integration and location-specific environmental insights.

USA-Wide Data Sources

  • OpenAQ API - National Air Quality Data
  • Open-Meteo API - Real-time Weather Across USA
  • USGS Water Services - National Streamflow Data
  • State Environmental Agencies - Regional Metrics
  • EPA AirNow - Federal Air Quality Standards
